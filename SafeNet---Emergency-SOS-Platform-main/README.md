# SOS Emergency Dashboard

**Real-time SOS Emergency Dashboard with Volunteer Tracking and Map Visualization**

---

## Project Overview

This project is a **real-time SOS Emergency Dashboard** that allows monitoring, managing, and responding to emergency alerts. It is built using **React**, **TypeScript**, **Zustand** for state management, and **Leaflet** for interactive map visualizations.  

The system is designed for volunteers and emergency response teams to track live emergencies, view alert details, and coordinate responses efficiently.

---

## Features

- **Live Emergency Map:** Dynamic markers for active alerts.  
- **Alert Filtering:** Filter alerts by type (medical, accident, fire, etc.).  
- **Responder Tracking:** Track active responders and average response time.  
- **Side Panel Details:** View detailed alert information including coordinates, people affected, and contact info.  
- **New SOS Alerts:** Easily create and submit new emergency alerts.  
- **Analytics:** View top locations, alerts by type, and time-based distributions.  

---

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS  
- **State Management:** Zustand  
- **Mapping:** Leaflet  
- **UI Components:** ShadCN/UI + Lucide Icons  
- **Version Control:** Git + GitHub
