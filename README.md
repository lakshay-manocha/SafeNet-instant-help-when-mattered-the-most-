# SafeNet-instant-help-when-mattered-the-most-
SafeNet is a rapid-response safety platform that delivers instant help when it matters most. With one-tap emergency alerts, real-time location sharing, and trusted contacts, it reduces response time and improves personal safety through fast, reliable, privacy-focused technology.
# SafeNet 

**Instant help when it matters the most.**

SafeNet is a **frontend-only Emergency SOS web application** focused on providing a fast, intuitive, and accessible user interface for triggering emergency alerts. The project demonstrates how safety-focused user experiences can be designed with modern frontend tools.

---

##  Project Type

* **Frontend only** 
* Built as a UI/UX-focused safety application
* Ideal for demos, hackathons, portfolios, and further backend integration

---

##  Features

*  Emergency SOS interface
*  Location-aware UI (extendable)
*  Trusted contact concept
*  Responsive, mobile-friendly design
*  Fast and lightweight frontend

### Prerequisites

* Node.js (v18+ recommended)

```bash

git clone https://github.com/your-username/SafeNet.git
cd SafeNet
```

```bash

```
